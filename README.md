## What it does

A Python script that runs automatically after US market close (22h30 Paris time, Mon–Fri) and sends a structured portfolio review by email. The analysis is generated by Claude (Anthropic API) with web search enabled — so it incorporates real market news, not just price data.

**Each nightly report covers:**
- Portfolio performance vs. S&P 500
- Position-by-position analysis with news context
- Key events of the day and their portfolio impact
- Risk flags and suggested follow-up actions

---

## How it works

```
GitHub Actions (cron: 21:30 UTC Mon–Fri)
  └── python portfolio_monitor.py --now
        ├── Fetch portfolio data (yfinance)
        ├── Call Claude API with web_search tool
        │     └── Claude searches news for each ticker
        ├── Generate structured markdown report
        └── Send via Gmail SMTP
```

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/nicolasraoulduval-ui/bourse-review.git
cd bourse-review
pip install -r requirements.txt
```

### 2. Configure environment

```bash
cp .env.template .env
```

Edit `.env`:

```env
ANTHROPIC_API_KEY=sk-ant-...    # console.anthropic.com
EMAIL_SENDER=your@gmail.com
EMAIL_PASSWORD=xxxx xxxx xxxx xxxx   # Gmail app password (16 chars)
EMAIL_RECIPIENT=your@email.com
```

> **Gmail app password:** Google Account → Security → 2-Step Verification → App passwords → Create

### 3. Configure your portfolio

Edit `portfolio_monitor.py` and update the `PORTFOLIO` dict:

```python
PORTFOLIO = {
    "AAPL": 10,   # ticker: number of shares
    "MSFT": 5,
    "MC.PA": 2,   # Paris-listed stocks: use .PA suffix
}
```

### 4. Test immediately

```bash
python portfolio_monitor.py --now
```

This runs the full pipeline immediately (bypasses the 22h30 schedule).

## Deployment on GitHub Actions

Add these secrets in **Settings → Secrets and variables → Actions**:

| Secret | Value |
|---|---|
| `ANTHROPIC_API_KEY` | Your Anthropic API key |
| `EMAIL_SENDER` | Gmail address |
| `EMAIL_PASSWORD` | Gmail app password |
| `EMAIL_RECIPIENT` | Destination email |

Create `.github/workflows/monitor.yml`:

```yaml
name: Market Monitor

on:
  schedule:
    - cron: '30 21 * * 1-5'   # 21:30 UTC = 22:30 Paris (winter)
  workflow_dispatch:            # Manual trigger

jobs:
  run:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.11'
      - run: pip install -r requirements.txt
      - run: python portfolio_monitor.py --now
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
          EMAIL_SENDER: ${{ secrets.EMAIL_SENDER }}
          EMAIL_PASSWORD: ${{ secrets.EMAIL_PASSWORD }}
          EMAIL_RECIPIENT: ${{ secrets.EMAIL_RECIPIENT }}
```
